# 研究で使えそうなコード置き場です。
## MATLABディレクトリ
このディレクトリにはMATLABで使用するmファイルが使用対象ごとに分類されています。
### FFTディレクトリ
このディレクトリにはMATLABで使用するmファイルのうちFFTに関係するファイルが分類されています。<br>
UIバージョンのほうが使いやすいとは思いますのでそちらを優先的にご使用下さい。
### Graphディレクトリ
このディレクトリにはMATLABで使用するmファイルのうちグラフ作成に関係するファイルが分類されています。<br>
毎回の初回起動時はグラフ描画に時間が掛かりますが、2回目以降はExcelより早いと思います。<br>
**なんといっても軽い！**
## Pythonディレクトリ
このディレクトリにはPythonで使用するpyファイルが使用対象ごとに分類されています。
### DICディレクトリ
このディレクトリにはPythonで使用するpyファイルのうちDICに関係するファイルが分類されています。<br>
いくつかライブラリに依存していますので確認してから実行して下さい。
### testディレクトリ
このディレクトリにはGithubの動作を確認するだけのために作成したpyファイルが分類されています。<br>
ただのゴミですので無視して下さい。
