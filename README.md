## ./CSV
### ./Suppot
このディレクトリにはクリップボードからCSVファイルを作成する際に使用するpy/au3/exeファイルが分類されています。<br>
- 3D計測-py.exe  
  3D計測.pyをPython環境がなくてもWindows環境があれば実行できるようにしたものです。クリップボードからとtxtファイルからのCSVファイルの作成ができます。
- 3D計測.py  
  3D計測-py.exeのソースコードです。Python環境があればこのファイルを実行することもできます。クリップボードからとtxtファイルからのCSVファイルの作成ができます。
- 3D計測(x64).exe  
  3D計測-py.exeがdll依存関係のエラーが発生したときに対策するためのものです。クリップボードからのCSVファイルの作成ができます。Windows7環境下では有用かもしれません。
- 3D計測.au3  
  3D計測(x64).exeのソースコードです。AutoIt言語を使用しています。多分これでは動きません。

## ./MATLAB
### ./FFT
このディレクトリにはMATLABで使用するmファイルのうちFFTに関係するファイルが分類されています。<br>
UIバージョンのほうが使いやすいとは思いますのでそちらを優先的にご使用下さい。
- FFT.m  
  一番初歩的なFFTツールです。全て設定系はコードに直打ちした上で実行して下さい。不適切な設定は検知できません。
- FFT_CSV_2point.m  
  FFT.mの機能はそのままにmmとμmの読み替えに対応したFFTツールです。数字のオーダが一致します。全て設定系はコードに直打ちした上で実行して下さい。不適切な設定は検知できません。
- FFT_CSV_2point_UI.m  
  FFT_CSV_2point.mの機能はそのままに対話型の設定入力を実装したFFTツールです。数字のオーダが一致します。全て設定系はコマンドラインの指示に従って入力して下さい。一部、不適切な設定を検知できます。
- FFT_UI.m  
  FFT.mの機能はそのままに対話型の設定入力を実装したFFTツールです。
### ./Graph
このディレクトリにはMATLABで使用するmファイルのうちグラフ作成に関係するファイルが分類されています。<br>
- graph_create.m  
  実験のデータをグラフ化するためのツールです。左側・右側/μm・mmのグラフをまとめて4つ出力することができます。毎回の初回起動時はグラフ描画に時間が掛かりますが、2回目以降はExcelより早いと思います。<br>
  <b>なんといっても軽い！</b>
- setting_assist.m  
  graph_create.mの設定を保持しておくようなCSVファイルの作成ツールを考えていますが、存在価値自体が微妙であることとやる気の問題から進んでいません。未完成です！
