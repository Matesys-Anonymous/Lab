# 研究で使えそうなコード置き場です。
## CSVディレクトリ
### Suppotディレクトリ
このディレクトリにはクリップボードからCSVファイルを作成する際に使用するpy/au3/exeファイルが分類されています。<br>
pyファイルとau3ファイルがexeファイルのソースコードです。<br>
pyファイルと準拠したexeファイルではクリップボードからとtxtファイルからのCSVファイルの作成ができます。<br>
au3ファイルと準拠したexeファイルではクリップボードからのCSVファイルの作成ができます。<br>
pyファイル系で実行した際にdll関係のエラーが発生する場合にはau3ファイル系の使用を検討して下さい。(Windows7など)
 
## Excelディレクトリ
このディレクトリには細々した計算で使用するExcelファイル(xlsm形式)が使用対象ごとに分類されています。<br>
マクロを使用していますのでセキュリティのエラーが出たら許可して下さい。
### ANSYS→グラフディレクトリ
このディレクトリにはANSYSで得られたデータをグラフに描画する際に使用するExcelファイルが分類されています。
#### 収縮ディレクトリ
主に収縮班のツールが格納されています。(拡張班は著作権の所在が曖昧なのでご要望があれば)
### DICディレクトリ
主にDICを行った際に使用するファイルが格納されています。
### Graphディレクトリ
主に実験データのグラフ作成を行う際に使用するファイルが格納されています。<br>
何度も言いますが、MATLABでの描画をおすすめしています。
 
## MATLABディレクトリ
このディレクトリにはMATLABで使用するmファイルが使用対象ごとに分類されています。
### FFTディレクトリ
このディレクトリにはMATLABで使用するmファイルのうちFFTに関係するファイルが分類されています。<br>
UIバージョンのほうが使いやすいとは思いますのでそちらを優先的にご使用下さい。
### Graphディレクトリ
このディレクトリにはMATLABで使用するmファイルのうちグラフ作成に関係するファイルが分類されています。<br>
毎回の初回起動時はグラフ描画に時間が掛かりますが、2回目以降はExcelより早いと思います。<br>
<b>なんといっても軽い！<b>
 
## Pythonディレクトリ

### DICディレクトリ
